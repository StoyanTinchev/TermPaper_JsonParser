\doxysubsection{Detailed Description}
A class that represents an array in JSON. 

The \doxylink{class_json_array}{Json\+Array} class encapsulates a vector of \doxylink{class_json_element}{Json\+Element} pointers and provides methods to manipulate and search the array, in addition to converting the array to a JSON-\/formatted string.
