\doxysubsection{Detailed Description}
Exception indicating an invalid boolean value in a JSON document. 

This exception is thrown when an attempt to parse a boolean value in a JSON document fails because the value is not \textquotesingle{}true\textquotesingle{} or \textquotesingle{}false\textquotesingle{}.
