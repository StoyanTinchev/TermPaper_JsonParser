\doxysubsection{Detailed Description}
This class is responsible for parsing JSON text. 

It reads a string of JSON text and constructs an in-\/memory representation of the JSON using {\ttfamily \doxylink{class_json_value}{Json\+Value}}, {\ttfamily \doxylink{class_json_array}{Json\+Array}}, and {\ttfamily \doxylink{class_json_object}{Json\+Object}} objects. The parsing process includes validation of the JSON text structure according to the JSON standard.
