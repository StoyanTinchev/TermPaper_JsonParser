\doxysubsection{Detailed Description}
The \doxylink{class_json_manager}{Json\+Manager} class provides functionalities to manage JSON operations. 

\doxylink{class_json_manager}{Json\+Manager} uses a \doxylink{class_json_parser}{Json\+Parser} to parse the JSON, and provides operations like opening, saving, validating, printing, searching, checking containment, setting, creating, deleting, and moving elements within a JSON.
