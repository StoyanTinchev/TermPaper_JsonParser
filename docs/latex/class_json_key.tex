\doxysubsection{Detailed Description}
A class representing a key in a JSON key-\/value pair. 

The \doxylink{class_json_key}{Json\+Key} class encapsulates a string key used in a JSON object. The key can be obtained or modified through accessor methods, and can be converted to a JSON-\/formatted string.
