\doxysubsection{Detailed Description}
A class that represents an object in JSON. 

The \doxylink{class_json_object}{Json\+Object} class encapsulates a vector of key-\/value pairs where the key is a \doxylink{class_json_key}{Json\+Key} pointer and the value is a \doxylink{class_json_element}{Json\+Element} pointer. The class provides methods to manipulate, search, and convert the object to a JSON-\/formatted string.
